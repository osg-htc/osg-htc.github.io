What is the AlphaFold3 Alignment Cache?

The AlphaFold3 Alignment Cache is a shared library of precomputed multiple sequence alignments (MSAs) distributed through the Open Science Data Federation (OSDF). Generating MSAs is one of the most time- and compute-intensive steps of an AlphaFold3 prediction. By caching these alignments once and serving them from OSDF data caches close to where jobs run, researchers can skip redundant search work and move straight to structure inference.

Read the documentation

Alignments are keyed by sequence hash, so any job requesting a protein whose MSA has already been computed — by you or by anyone in the community — retrieves it directly from the cache instead of recomputing it.

Why cache alignments?

MSA generation for a single query can take from minutes to many hours of CPU time depending on sequence length and database size. Across a community running large batches of predictions, the same well-studied proteins are searched again and again. The alignment cache removes that duplicated effort:

  • Faster predictions — jobs that hit the cache skip the search stage entirely and start inference immediately.
  • Lower compute cost — capacity that would have gone to repeated alignment searches is freed for new science.
  • Reproducibility — everyone using a cached alignment runs against the exact same MSA input.
  • Community scale — every contributed alignment makes the next researcher's run faster.

How can I use the cache?

The cache is consumed the same way as any other dataset on the OSDF. Jobs submitted from an OSG-Operated Access Point request alignments through the OSDF namespace, and HTCondor handles transfer from the nearest cache. When an alignment is not yet cached, the workflow computes it and writes it back so the next request is a cache hit.

The alignment cache is a good fit if you are

  • Running AlphaFold3 (or other MSA-based structure tools) on many sequences.
  • Re-predicting structures for proteins that are common across studies.
  • Part of a collaboration that repeatedly works with the same organisms or protein families.

How can I contribute alignments?

Community contributions are what make the cache valuable. When your jobs generate new MSAs, you can publish them back to the shared namespace so others benefit. Contributed alignments are deduplicated by sequence and validated before being made broadly available.

Institutions or collaborations that would like to contribute alignments or host cache capacity should contact support@osg-htc.org.
